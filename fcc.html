Flower Crown Chaos

  • Genre: 2-D Action Roguelite
  • Engine: Unity
  • Project date: May 2024 - Present
  • Project URL: https://ndlovelace13.itch.io/flower-crown-chaos
  • Github Repo: https://github.com/ndlovelace13/FCC

Repel the replicant horde and let your creativity shine with thousands of powerful crowns!

Flower Crown Chaos is my first solo project under the No Rush Games title. What began as proof to myself that I could motivate myself to make a game steadily grew throughout the summer into a roguelite experience with multiple avenues of progression and player expression. Consistently working on this project significantly improved my coding abilities, as I put object-oriented design pillars into practice with my flexible crown construction algorithm. Additionally, I learned how to serialize game saves using json, gained more experience with state machines through creation of multiple enemy types, and worked with a core game controller to store and use variables across the many menus I created. As I continue to expand the project, I plan to integrate more sound effects using Wwise, build flexible challenges and floral mastery systems, and hopefully add enough content to release the game on Steam. Check out the source code to see exactly what makes this game tick!
