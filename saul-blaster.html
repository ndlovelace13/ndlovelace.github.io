Saul Blaster

  • Genre: Typing Shoot 'em Up
  • Engine: Custom Engine built with XNA
  • Project date: December 2023
  • Project URL: https://ndlovelace13.itch.io/saul-blaster

Put your typing skills to the test and show your brother who’s the better lawyer in Saul Blaster!

Saul Blaster is a fairly basic title built as a showcase for my Game Engine class. For what it may lack in depth and presentation, this title makes up for in the programming work that went into the construction of the engine. Throughout the semester, we were tasked with using the Microsoft XNA Framework to build core engine mechanics such as rendering pipelines, physics systems, and GUI components from scratch. This class and project brought a new appreciation for the work that goes into pre-built engines such as Unity. Another highlight of this project was the typing input system I had to build from scratch - it wasn't pretty code but it got the job done.
