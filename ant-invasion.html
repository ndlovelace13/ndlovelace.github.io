AI: Ant Invasion

  • Genre: 2-D Idle Game
  • Engine: Unity
  • Project date: October 4-7, 2024 - Ludum Dare 56
  • Project URL: https://ndlovelace13.itch.io/ai-ant-invasion
  • Github Repo: https://github.com/ndlovelace13/ludum-dare-56

Control the evolution of an ever-hungry ant colony!

AI: Ant Invasion is a title made with a fellow CS student for the Ludum Dare 56 game jam. The theme for this jam was "Tiny Creatures" and we decided to design a game based around ants eating their way through larger and larger materials as they evolve. The two core programming components for this project were state machines, which controlled all ant behavior, and a grid algorithm, used to determine which nodes would get consumed when an ant arrived to eat. We also implemented a queue system for each feeding location and queen ant - this was used to ensure ants would properly take their turn and avoid multithreading issues like deadlock. All systems were built with flexibility to allow integration with the various mutations users can invest in to augment the abilities of their ants.
